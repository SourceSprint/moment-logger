# Moment Logger
Fancy terminal logging with timestamps

```js
const logger = require('@boxpositron/moment-logger')

# Display hello world

logger.log('Hello world')

# Display world object

const worldObject = {
    earth: ['Water', 'Earth', 'Air', 'Fire']
}

logger.log(worldObject)

```
